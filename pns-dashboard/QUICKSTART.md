# PNS Admin Dashboard - Quick Start

## ⚠️ Important for Windows Users
You seem to have PowerShell script execution disabled.
Run the following commands using **Command Prompt (cmd.exe)** or prefix with `cmd /c` in PowerShell.

### 1. Install Dependencies
```cmd
npm install
```
*Note: We have already attempted to install the core dependencies for you.*

### 2. Start the Application
To start the development server:
```cmd
npm run dev
```

### 3. Open in Browser
Once running, open [http://localhost:5173](http://localhost:5173) in your browser.

## Features
- **Notifications**: Full management interface.
- **Multi-language**: EN/ES/FR support.
- **Premium UI**: Glassmorphism and animations.
